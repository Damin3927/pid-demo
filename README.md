# PID制御 インタラクティブ入門

1関節ロボットアームを題材に、PID制御を **P → PD → PID** と一段ずつ積み上げ、さらに
**インピーダンス／アドミタンス制御**（力・接触の制御）まで学ぶインタラクティブ Web デモです。
各項が「いま何をしているか」をリアルタイムのトルク内訳バーで可視化し、各段階で起きる現象を
体験できます。

## 学べること

### 位置制御（PID）

| ステップ | 体験する現象 |
| --- | --- |
| **P制御** | 比例ゲインだけ。速いほど振動し、重力のせいで目標手前で止まる（**定常偏差**） |
| **PD制御** | 微分でブレーキ。振動が収まり大きな Kp が使える。だが定常偏差は残る |
| **PID制御** | 積分が一定トルクを作り出し、**定常偏差が消える**。ワインドアップも体験可能 |
| **自由実験** | すべてのつまみを自由に。速さ・安定性・正確さのトレードオフを探る |

### 力・接触の制御（インピーダンス／アドミタンス）

「外力（押す力）」スライダーで実際にアームを押し、しなやかな従い方を比べられます。

| ステップ | 体験する現象 |
| --- | --- |
| **インピーダンス制御** | トルクで「仮想バネ＋ダンパ」を作る。力は**測らず**、変位から力を出す。押すと `たわみ ≈ F/K` の**コンプライアンス** |
| **アドミタンス制御** | 双対。外力を**測って**「仮想質量＋ダンパ」の運動を作り、内側の硬い位置ループで追従。押すと「質量 Mv」のように動く |

## 物理モデル

ピボットまわりの剛体回転として、角度依存の重力負荷を外乱に持つ系を扱います。

```
J·θ'' = τ_motor + τ_ext − b·θ' − m·g·L·cos(θ)

PID         : τ_motor = clamp(Kp·e + Ki·∫e dt + Kd·de/dt,  ±τ_max)
インピーダンス : τ_motor = clamp(K·(θd−θ) − D·θ̇ + ĝ(θ),       ±τ_max)
アドミタンス   : Mv·θ̈ᵣ + Dv·θ̇ᵣ + Kv·(θᵣ−θd) = τ_ext      (仮想ダイナミクス)
              τ_motor = clamp(Kp_in·(θᵣ−θ) + Kd_in·(θ̇ᵣ−θ̇) + ĝ(θ), ±τ_max)
```

`m·g·L·cos(θ)`（重力負荷）は目標角度でもゼロにならないため、P / PD では誤差を残さないと
釣り合わず、これが定常偏差として現れます。積分項がこの保持トルクを肩代わりして偏差を消します。

インピーダンス／アドミタンスでは、外乱を打ち消す位置制御とは逆に「**動きと力の関係**」そのものを
設計します。インピーダンスは変位 → 力（バネ）で外力 `τ_ext` を**計測しない**のに対し、アドミタンスは
力 → 運動（質量）で `τ_ext` を**計測して使う**——同じアームで双対の性質を比較できます。`ĝ(θ)` は
重力補償の前回送り（チェックで ON/OFF）。

- 数値積分: セミインプリシット（シンプレクティック）オイラー、固定ステップ `dt = 2ms`
- 微分項は測定値（角速度）に対して作用させ、目標変更時の微分キックを回避
- 積分にはアンチワインドアップ（条件付き積分）を実装。PID/自由実験で ON/OFF 切替可能

## 開発

```bash
pnpm install
pnpm dev        # 開発サーバ
pnpm build      # 型チェック + 本番ビルド
pnpm preview    # ビルド結果のプレビュー
pnpm typecheck  # 型チェックのみ
```

## 技術スタック

Vite + React + TypeScript + Tailwind CSS v4。可視化（アーム・応答グラフ）は依存を増やさず
Canvas に直接描画しています。シミュレーションのコアは `src/sim/pid.ts`（純粋関数）に分離。

## 構成

```
src/
  sim/
    pid.ts            物理 + PID の純粋なステップ計算
    usePidSim.ts      PID 用 React フック（requestAnimationFrame・指標計算込み）
    control.ts        インピーダンス／アドミタンスの純粋なステップ計算
    useControlSim.ts  力制御用 React フック（外力入力・指標計算込み）
    config.ts         プラント定数・ゲイン範囲
  components/          ArmCanvas / ResponseChart / GainSlider / TermBar /
                      PidLab / ControlLab / ControlTermBar / ControlSimControls / ...
  steps/steps.tsx      各ステップの解説文と既定パラメータ
  App.tsx              ステッパーと画面遷移
```
